BGA(Ball Grid Array) 검사 프로그램 권장 사항

1. 개요
BGA 볼 사이즈, 개수, 피치 등 정밀 검사에서는 고해상도 특징 추출이 가능한 비전 모델과 전통 영상처리를 조합하는 것이 효과적입니다.

2. 모델 추천
① 산업용 전용 모델
- YOLOv8 / YOLOv9 + 고해상도 입력
  장점: 빠르고 가벼움. 객체 검출과 위치 정보 제공.
  활용: 볼 위치 검출 후 OpenCV로 직경·피치 계산.

- Detectron2 (Mask R-CNN)
  장점: 픽셀 단위 세그멘테이션으로 정밀 측정에 유리.
  단점: YOLO 대비 속도 느림.

- MMDetection
  다양한 최신 모델 실험에 유리.

② 전통 영상처리 기반 (정밀 측정)
- OpenCV + Hough Circle Transform
  규칙적인 원형 배열 인식에 강함. 조명·배경 조건이 좋으면 매우 정밀함.

- Edge Detection + Blob Analysis
  외곽선 추출 후 개수 및 거리 계산으로 활용.

③ 하이브리드 접근 (권장)
- YOLO 계열로 볼 후보 영역 검출 → OpenCV로 정밀 치수 계산
  이 방식은 속도와 정확도 균형이 좋음.

3. 추천 조합 예시 (파이프라인)
- YOLOv8-seg로 볼 마스크 생성
- OpenCV로:
  1) 볼 중심 좌표 계산
  2) 볼 직경 측정 (외곽 픽셀 기반)
  3) 피치 측정 (인접 중심 간 거리)
  4) 볼 개수 카운트 및 누락 검출

4. 장비·환경 권장사항
- 카메라: 고해상도(≥12MP), 텔레센트릭 렌즈 사용 권장 (왜곡 최소화)
- 조명: 링 라이트 또는 도미네이트 균일 조명
- 이미지 전처리: CLAHE 적용 시 인식률 향상

5. 추가 제안
원하시면 YOLO + OpenCV 하이브리드 BGA 검사 파이프라인의 예제 코드를 제공해 드리겠습니다. 데이터셋 준비 후 바로 테스트 가능하도록 구성할 수 있습니다.
